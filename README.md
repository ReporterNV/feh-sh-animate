# feh-wallpaper-animator
Turn a .gif file into an animated wallpaper with few system resources

Pre-requisites
===
ImageMagick  
feh  

Installation
===
Run 
```
$ cd /path/to/gif && convert pic.gif -coalesce pic.png
$ cd /path/to/gif && chmod +x animate-wallpaper.sh
```
